## Tech Stack

### Core

- Swift 6
- SwiftUI
- iOS 18+

### State Management

- @Observable for view state
- Manual dependency injection (initializers, no libraries)

### Networking

- NetKit (https://github.com/macward/NetKit)

### Persistence

- Keychain for tokens and sensitive data

### Navigation

- NavigationStack with NavigationPath (primary)
- AppRouter (https://github.com/Dimillian/AppRouter) for complex flows

### Media

- AsyncImage for remote images

### Logging

- OSLog (native)

### Not Using (yet)

- SwiftData / Core Data
- Feature flags / Remote config
- Third-party analytics
